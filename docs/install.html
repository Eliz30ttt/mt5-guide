Install MT5 & Place Your First Trade

Follow this simple checklist to get your platform ready and execute your first demo trade safely.

Step 1 — Download MT5

  1. Open your broker dashboard
  2. Find “Platforms” or “Download MT5”
  3. Install on desktop or mobile
  4. Launch the app

Step 2 — Login to Your Broker Account

  1. Open MT5
  2. Select Login to Trade Account
  3. Enter login ID
  4. Enter password
  5. Select your broker server

Step 3 — Open a Chart

  1. Open Market Watch panel
  2. Right-click symbol (EURUSD, Gold, Oil, etc.)
  3. Select Chart Window
  4. Choose timeframe (M5, M15, H1)

Step 4 — Place First Demo Trade

  1. Click New Order
  2. Choose lot size (start very small)
  3. Set stop loss
  4. Set take profit
  5. Click Buy or Sell
Pro tip: Always start on a demo account first. Practice execution and risk control before going live.

Risk Rule for Beginners

  • Risk max 1–2% per trade
  • Never trade without stop loss
  • One trade at a time when starting
  • Journal every trade
